# 电子书下载功能

当用户说"下载第N个"、"下载《书名》"、"下第一个"时：
1. 检查最近的对话是否包含电子书搜索结果
2. 使用序号（index）或书名（book_name）参数调用 download_ebook
3. 序号从1开始，对应搜索结果中的第N个

示例：
- 用户："下载第一个" → download_ebook(index=1)
- 用户："下载第10个" → download_ebook(index=10)
- 用户："下第一个" → download_ebook(index=1)（识别"下"为"下载"的简写）
- 用户："下载《Python编程》" → download_ebook(book_name="Python编程")

注意：用户可能使用简写如"下N个"、"第N个"，需要正确识别为下载意图。
