# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-21

### Added
- 首次发布移动AI助手应用，支持iOS和Android平台
- 实时聊天功能，支持文本、Markdown和数学公式渲染
- 语音通话功能，基于WebRTC实现实时音频通信
- 对话管理功能，支持创建、搜索和管理多个对话
- 服务配置功能，支持Dify和DIY自定义服务接入
- 版本管理系统，自动检查应用更新和版本兼容性
- 设置页面，支持服务器配置、模型选择和通用设置
- 字体大小调节和显示偏好设置
- 关于页面，包含许可证和隐私政策

### Changed
- 优化WebSocket连接稳定性
- 改进音频录制和播放性能

### Security
- 实现安全的WebSocket通信
- 添加用户数据加密存储
